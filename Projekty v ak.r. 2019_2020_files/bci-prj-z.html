Tato práce má za cíl implementovat aplikaci umožňující použití programu Git a jeho rozšíření Git LFS a Git Annex na operačním systému Android. Poslouží zejména vývojářům k usnadnění práce s Git a velkými soubory na této platformě. Uživatelské rozhraní je proto navrženo jako maximálně transparentní, za účelem efektivního řešení problémů vznikajících při použití Git.
